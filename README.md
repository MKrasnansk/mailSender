## Description

[Nest](https://github.com/nestjs/nest) Mail Sender to mailtrap.io. with MailerModule.
Easy and pretty.

Jednoduchy mail-sender na odosielanie sprav na Váš mail.

## Installation

```bash
$ npm install
```
## .env
```bash
rename .env.example to .env and fill data
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
